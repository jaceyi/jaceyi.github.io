很多新手都想搭建一个属于自己的网站/博客，自己搞服务器的话又得花钱还需要对后端以及服务器方面有相关的了解。 本篇讲述使用 GitHub Pages 和 Jekyll 搭建一个属于自己的静态网站/博客。

GitHub 就不用详细介绍了。GitHub Pages 则是 GitHub 上的一项功能，可以将仓库里的内容转成网页展示出来，并分配一个专属域名

Jekyll 的作用是将纯文本转换为静态博客网站，支持 Markdown、Liquid、HTML。（Jekyll 中文网）
